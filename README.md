# МАУП - Сайт дистанційного навчання

Сучасний лендінг для дистанційного навчання Міжрегіональної Академії управління персоналом.

## Швидкий старт

```bash
npm install
npm run dev
```

Сайт відкриється автоматично за адресою `http://localhost:3000`

## Збірка

```bash
npm run build   # продакшн (single-file HTML)
npm run preview # попередній перегляд збірки
```

## Структура проекту

```
├── public/
│   ├── favicon.svg
│   └── images/          # статичні зображення
├── src/
│   ├── components/
│   │   └── LandingPage.tsx
│   ├── App.tsx
│   ├── index.tsx
│   └── index.css
├── index.html
└── vite.config.ts
```

## Технології

- **React 18** + **TypeScript**
- **Tailwind CSS** — стилізація
- **Framer Motion** — анімації
- **Lucide React** — іконки
- **Vite** + **vite-plugin-singlefile** — збірка в один HTML файл
