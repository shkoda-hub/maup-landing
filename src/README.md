
# МАУП - Сайт дистанційного навчання

Сучасний лендінг для дистанційного навчання Міжрегіональної Академії управління персоналом.

## 🚀 Швидкий старт

### Встановлення

```bash
npm install
```

### Запуск проекту

```bash
npm run dev
```

Сайт відкриється автоматично за адресою `http://localhost:3000`

### Збірка для продакшену

```bash
npm run build
```

### Попередній перегляд збірки

```bash
npm run preview
```

## 📁 Структура проекту

```
├── components/
│   └── LandingPage.tsx    # Головний компонент сторінки
├── public/
│   └── images/            # Папка для зображень (додайте свої фото)
├── App.tsx                # Кореневий компонент
├── index.tsx              # Точка входу
├── index.css              # Глобальні стилі
└── index.html             # HTML шаблон
```

## 🎨 Технології

- **React 18** - UI бібліотека
- **TypeScript** - типізація
- **Tailwind CSS** - стилізація
- **Framer Motion** - анімації
- **Lucide React** - іконки
- **Vite** - збірка проекту

## 📝 Налаштування

### Зміна кольорів

Відредагуйте CSS змінні в `index.css`:

```css
:root {
  --color-primary: #1E40AF;      /* Основний синій */
  --color-accent: #F59E0B;       /* Акцентний золотий */
  /* ... інші кольори */
}
```

### Додавання зображень

1. Додайте зображення в папку `public/images/`
2. Використовуйте їх у коді: `<img src="/images/your-image.jpg" />`
